# Our K-drama Date 💗

A soft, playful date-planning page. Single HTML file, hosted on GitHub Pages.

## How it works
- Step 1: RSVP (the "No" button runs away).
- Step 2: Pick a date and time.
- Step 3: Choose an activity pairing.
- Step 4: Tap "send" — your email app opens with a playful confirmation pre-filled (including a mention of the chocolates and flowers on their way).

No EmailJS, no API keys, no signups — pure static HTML.
